Testing

  • Follow the instructions under Building from Source
    • Make sure nodejs is in "program files"
    • Navigate to the community-app directory in the git shell before doing npm install -g bower and all other commands
    • Proceed with the rest of the steps
  • Type grunt serve to check the results on the demo, it will start a http server that is called to run tasks and serve files, it will automatically update when a file is saved
  • An easier option is to just open the index file with Firefox
