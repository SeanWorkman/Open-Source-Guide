Adding to Path Variable

Find the Path

  • Find the program in your files
  • Right click on the program folder and go to "properties" (Java, Python) copy the "Location"
    • Java- open the "jdk1.8.0_20" folder (numbers change depending on version) in the "Java" folder to get the Path add \bin to the end when adding to Path Variable (C:\Program Files\Java\jdk1.8.0_20\bin)
    • Python- C:\Python27 (changes depending on version and location in files)

Windows

  • Control Panel
  • System
  • Advanced System Settings on the left
  • Environment Variables...
  • Find the variable "Path", click on it
  • Edit...
  • In the "Variable value:" text box do not delete anything, go to the end, add a ";" and paste the path for the program (Java, Python, etc...)
  • For Python also add ;C:\Python27\Scripts ("Python27" should be whatever version of Python you downloaded)

Mac/Linux

  • Open the Terminal
  • Type export PATH=$PATH:'the_program's_path'
  • Type echo $PATH to check the path
