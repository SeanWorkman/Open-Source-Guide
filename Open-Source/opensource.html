Open Source

Open Source software is a rapidly growing option for everyday people that want to develop a program, large companies, and non-profit organizations, meaning that there is a plethora of options to choose from. Everyone has their own motivation for getting involved with Open Source. It is a great way to build your resume and skills and it is a great way to give back on a global scale instead of volunteering at a local soup kitchen you can help provide much needed help for the less fortunate around the world by contributing to a world-wide non-profit organization.

Open Source products include:

  • Linux
  • Notepad++
  • Audacity
  • Wordpress
  • Firefox
  • Apache
  • MySQL

This is not a guide on where to go to figure out how to find projects, it is a step by step guide on getting set up, where and how to find the project that is best for you, once you find the right project it shows how to find and complete tasks, and then submit those contributions.

Setup

  • Install Linux CentOS 7 by following the video, most Open Source projects use Linux
  • Set up a github account

Finding the Right Project and Tasks

In the world of Open Source there are a lot of projects to choose from, some being bigger or more complex than others, here are a few suggestions:

  • Sahana Eden- larger project with a good amount of outside libraries which can get confusing, tasks are very easy to find in the code
  • Mifos X- larger project that requires a reasonable amount of knowledge about financial terms
  • OpenStack- medium sized project with a slightly more complex work flow
  • OpenHatch- database of countless Open Source projects, search by language, project, and difficulty

Submitting Changes

After you have fixed an issue it is time to submit your changes to the project. Every project has a different way of taking submissions, this is called the workflow. Some workflows are more complex than others.
  • Sahana Eden
  • Mifos X
  • OpenStack
